# 🏦 Bank Management System (Java)
📌 Overview

A console-based Bank Management System built using Java.
This project demonstrates core Object-Oriented Programming (OOP) concepts and simulates real-world banking operations like account creation, login, deposit, withdrawal, and transaction tracking.

✨ Features
🆕 Create new bank account
🔐 Secure login (Account Number + PIN)
💰 Deposit money
💸 Withdraw money
📄 View account details
📊 Transaction history tracking
📋 Menu-driven interface
🧠 Concepts Used
✔️ Abstraction (Interface: BankOperations)
✔️ Inheritance (Base class: Bank)
✔️ Encapsulation (Private variables & methods)
✔️ Java Collections (ArrayList)
✔️ User Input Handling (Scanner)
🏗️ Project Structure
Bank_System/
│── BankSystem.java   # Main class (entry point)
│── Account class     # Handles user account logic
│── Bank class        # Base class with bank details
│── BankOperations    # Interface for operations
